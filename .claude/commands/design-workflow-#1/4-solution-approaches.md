## Command

Propose 4-6 fundamentally different solution approaches for the problem framing.

Each approach should optimize for a different user priority:
- Speed, Clarity, Trust, Control, Effort, or Accuracy

For each approach, provide:
- Core concept (2-3 sentences)
- How it works (bullet points)
- Trade-offs table (Pros vs Cons)
- Best suited for which user type
